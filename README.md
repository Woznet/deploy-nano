# deploy-nano

## **Ubuntu**
- edit config files
- install apps like powershell, gh, nvm, npm
- build [nano-7.2][4]
- add [nanorc syntax][1]

***

## **Windows**
- download [nano-win][2]
- add [nanorc syntax][1]
